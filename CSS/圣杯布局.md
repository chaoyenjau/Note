```html
<style>
  $left-w: 300px;
  $right-w: 300px;
  
  #container{
    padding : 0 $right-w 0 $left-w;
  }
  #mid, #left, #right{
    float: left;
    position: relative;
  }
  #mid{
    width: 100%;
  }
  #left{
    width: $left-w;
    margin-left: -100%;
    left:-$left-w;
  }
  #right{
    width: $rihgt-w;
    margin-left: -$right-w;
    right: -$right-w;
  }
</style>
<div id='container'>
  <div id='mid'>mid</div>
  <div id='left'>left</div>
  <div id='right'>right</div>
</div>
```
